# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-07

### Added
- `meowdel chat` — interactive chat session with cat personalities
- `meowdel ask` — one-shot question mode
- `meowdel brain` — Brain knowledge base commands (list, search, new, get, graph, agent)
- `meowdel config` — interactive setup for API keys, personality, and base URL
- `meowdel login` — open meowdel.ai in browser to retrieve API key
- `meowdel personalities` — list available cat personalities
- Bring-your-own Claude API key support (direct Anthropic access)
- Self-hosted instance support via `--url` flag
- Brain context injection into chat sessions with `-b` flag
